OpenRouter Coding Agents TOP10（月度 Token 排行）

刷了一下 OpenRouter 的 Coding Agents 月榜。
前两名就是上次聊的"养虾又养马"——Hermes + OpenClaw。

1️⃣ Hermes Agent · 629B
Nous Research 出品，开源自主 Agent。跨会话记忆 + 40+ 工具 + 子 Agent 并行。直接登顶。

2️⃣ OpenClaw · 154B
飞书生态 + 50+ 消息平台。ClawHub 4.4 万技能，370K GitHub Star。

3️⃣ Kilo Code · 136B
VS Code/JetBrains/CLI 三端。Orchestrator 模式拆出 planner/coder/debugger 子 Agent。150 万用户，800 万美金种子轮。

4️⃣ Claude Code · 56.7B
Anthropic 官方。读全库、跨文件改、跑测试、迭代修复。大厂里最稳的编码 Agent。

5️⃣ pi · 37.5B
"编码 Agent 很多，但这个是你的。"自带 API Key，CLI 原生，主打个性化和隐私。

6️⃣ Roo Code · 20B
VS Code 多 Agent 团队，架构/编码/调试/审查分工。

7️⃣ Cline · 19.1B
IDE 自主 Agent。翻代码、改文件、跑命令、浏览器自动化。Roo/Kilo 都 fork 自它。

8️⃣ Lemonade · 12.9B
专做 Roblox 游戏的 AI。垂直场景。

9️⃣ Codex · 7.37B
OpenAI 官方。最近加了 Chrome 插件控制浏览器。

🔟 Zed Editor · 4.86B
Rust 写的高性能编辑器。AI 辅助 + 低延迟协作。

观察：

🟣 前 10 里 9 个开源，只有 Claude Code 闭源。

🟣 Kilo → Roo → Cline 是一条 fork 链，各走各路但底层共享 DNA。

🟣 Hermes + OpenClaw 同一台机器跑双 Agent 是真实方案，一管国际一管飞书。

🟣 前 5 名 4 个是通用 Agent，Agent 正在走出编辑器。
