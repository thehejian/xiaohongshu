Tweet 4:
You think you're using AI.
AI is using you.

ADHD symptoms:
• Can't focus for more than 5 min
• Habitual Ctrl+TAB
• 8 open chats, none finished

Tweet 5:
3 antidotes:

1. Long-context models over fragmented queries
Give one model the full context. 5 conversations → 1.

2. Task bundling
One task = one tool. No cross-tool switching.

3. Think offline first
Write 3 key questions before opening AI. Reduces useless chats by 80%.

Tweet 6:
Tools should liberate you.
Not turn you into a machine.

Reduce AI switching frequency.
Return to deep work.
