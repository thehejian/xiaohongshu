---

### 它能做什么？

**🤖 11 个专业 Agent 各司其职**

Sisyphus 总指挥 + Hephaestus 深度执行，Oracle 策略分析 + Librarian 知识管理，Explore 调研探索。每个 Agent 配最优模型，不用手动调。

**👥 Team Mode — 多 Agent 并行作战**

1 个主控 + 8 个并行成员。`hyperplan`：5 个批评家撕碎你的方案。`security-research`：3 猎人 + 2 PoC 并行审计。

**⚡ ultrawork — 一个字，干到底**

输入 `ultrawork`，所有 Agent 激活。不达目标不停止。

**🛠 更多能力**

LINE#ID 哈希编辑 · LSP 集成 · AST-Grep · Ralph 自动循环 · 内置 MCP · 规则注入
