# 中国AI六小虎现状盘点 — 智谱 vs MiniMax vs 月之暗面 vs 百川 vs 零一万物 vs 阶跃星辰

> 小红书风格文章 · 2026.06.03 · 正文 ~900 字

---

## 📖 文章正文

2026年，中国AI大模型"六小虎"格局发生剧变。从最初的百模大战中厮杀出来的六家公司——智谱AI、MiniMax、百川智能、月之暗面（Kimi）、零一万物、阶跃星辰——正走出截然不同的道路。智谱和MiniMax已在港股上市，月之暗面估值突破200亿美元，阶跃星辰冲刺IPO，百川押注医疗赛道，零一万物加速出海。本文带你一张图看懂最新格局。

### 🏢 智谱 AI — 政企之王

| 维度 | 数据 |
|------|------|
| 创始人 | 张鹏（清华系） |
| 成立 | 2019年（清华孵化） |
| 上市 | 2026年1月港股上市 |
| 市值 | ~4400亿港元（约¥3800亿） |
| 模型 | GLM系列 |
| 客户 | 12000+企业，8000万+终端 |
| 2025营收 | ¥7.24亿 |
| 标签 | 全球大模型第一股 |

智谱走的是"政企驱动型"路线，聚焦政府、央国企、金融、能源等高安全需求场景。深嵌信创生态，通过API+私有化部署双轮驱动。2026年1月在港交所挂牌，成为"全球大模型第一股"，奠定了行业标杆地位。

### 🎮 MiniMax — 全球化多模态

| 维度 | 数据 |
|------|------|
| 创始人 | 闫俊杰（前商汤） |
| 成立 | 2021年12月（上海） |
| 上市 | 2026年1月港股上市 |
| 市值 | ~2500亿港元（约¥2200亿） |
| 模型 | MiniMax M3 / M2.7 |
| 收入 | 2025前三季度$5343万 |
| 用户 | 2.36亿个人 + 21万企业 |
| 标签 | 全球化AGI第一股 |

MiniMax走出了一条独特的C端全球化道路。旗下Talkie是全球AI伴侣类头部应用，海螺AI视频生成用户数达4235万。海外收入占比超70%，多模态产品矩阵覆盖文本、语音、视频、音乐。

### 🌙 月之暗面（Kimi）— C端超级入口

| 维度 | 数据 |
|------|------|
| 创始人 | 杨植麟（清华叉院） |
| 成立 | 2023年 |
| 估值 | ~200亿美元（¥1400亿） |
| 模型 | Kimi K2.6（1T参数MoE） |
| ARR | 超2亿美元（2026.4） |
| 累计融资 | 376亿+人民币 |
| 标签 | C端用户规模最大 |

Kimi是六小虎中C端用户规模最大的公司。最新K2.6模型首发即开源，在多项全球基准测试中持平GPT-5.4。中国移动参投最新20亿美元融资轮，未来C端付费增长值得关注。

### 🏥 百川智能 — 医疗垂类深耕

| 维度 | 数据 |
|------|------|
| 创始人 | 王小川（前搜狗CEO） |
| 成立 | 2023年 |
| 估值 | ~200亿人民币 |
| 方向 | 医疗AI（B端） |
| 融资 | 50亿A轮（2024） |
| 现金储备 | 30亿元 |
| 标签 | 医疗大模型领跑者 |

百川几乎是六小虎中最独特的——它放弃了C端消费者战场，全面转向企业级医疗市场。医疗垂类大模型需要长期数据积累、监管审批和医院渠道，这是短期烧钱买不到的东西。王小川计划2027年启动IPO。

### 🌏 零一万物 — 出海先锋

| 维度 | 数据 |
|------|------|
| 创始人 | 李开复 |
| 成立 | 2023年 |
| 估值 | ~10亿美元 |
| 模型 | Yi系列 |
| 2024营收 | 超1亿元 |
| 产品 | 万智企业大模型平台 |
| 标签 | 海外业务起步最早 |

零一万物的气质介于学术与商业之间。海外业务在东南亚、中东市场有一定客户基础，这是国内纯B端厂商不具备的能力。核心产品"万智企业大模型平台"已落地中国移动等标杆客户。正在Pre-IPO阶段。

### 🚀 阶跃星辰 — 端侧终端之王

| 维度 | 数据 |
|------|------|
| 创始人 | 姜大昕（微软亚洲研究院） |
| 成立 | 2023年（上海） |
| 估值 | ~200亿人民币 |
| 融资 | 近25亿美元 |
| 装机量 | 4200万+终端设备 |
| 方向 | 多模态 + AI终端 |
| 标签 | 端侧AI领跑者 |

阶跃星辰的技术标签是多模态能力。其模型已在OPPO、荣耀、中兴等品牌中装机超4200万台，日均服务近2000万人次。在汽车端，与吉利合作的AgentOS系统已搭载于银河M9车型。上海国资和头部VC同时押注。
