## 📖 文章正文
编码 Agent 界新王登基 ⚔️
它叫 **Pi**，来自 Earendil Inc.（Pi.dev）
### 为什么说 Pi 是下一代 Agent？
别的 Agent 告诉你「我有什么」
Pi 告诉你「你可以自己造」🛠️
### 核心亮点
**1️⃣ 极简但极度可扩展**
- 没有内置 MCP、子代理、plan 模式
- 但你可以用 Extension 自己造
- 缺什么？直接让 Pi 帮你写
**2️⃣ 15+ 模型供应商**
- Anthropic / OpenAI / Google / Azure / Mistral / Groq / Cerebras / xAI / Kimi / MiniMax / OpenRouter / Ollama...
- 随时切换：`/model` 或 `Ctrl+L`
**3️⃣ 树状会话历史**
- 会话存成树形，任意跳转分支
- 支持 HTML 导出和 GitHub Gist 分享
**4️⃣ 真正的上下文工程**
- AGENTS.md 项目指令
- SYSTEM.md 替换系统提示词
- Skills 按需加载，不爆 context
- 支持自定义 compaction 策略
### 安装方式
```bash
curl -fsSL https://pi.dev/install.sh | sh
# 或 npm/pnpm/bun
npm install -g @earendil-works/pi-coding-agent
```
### 它适合谁？
- 不想被工具绑架工作流的开发者
- 想要完全掌控 Agent 行为的极客
- 需要高度定制化 CI/CD 流程的团队
**2026，Agent 的玩法变了 🔥**
不再是「选哪个 Agent」
而是「你想要什么样的 Agent」
**Pi，就是你的答案。**
