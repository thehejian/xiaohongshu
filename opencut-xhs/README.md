剪映太臃肿？试试这个开源平替 👇

**OpenCut** 是开源的 CapCut（剪映国际版）替代品，40K+ GitHub 星，隐私优先，轻量快速。

### 🎬 视频剪辑全套

多轨时间轴、裁剪分割、字幕添加、变速、转场、滤镜、关键帧动画。导出 4K 视频，无画质损失。

### 🤖 AI 黑科技

- **声音克隆** — 复制你的声音，相似度达 99%
- **视频翻译** — 一键翻译，短剧出海必备
- **人声分离** — 清空背景音，提取纯净人声
- **AI 去水印** — 一键消除 LOGO 和物体

### 🖼️ 图片处理也包了

AI 抠图、图片超清修复（老照片变 4K）、AI 动漫化、图片翻译（不损原图，自动换文字）。

### 💡 亮点

- **完全开源** — MIT 协议，代码透明
- **隐私优先** — 不采集数据，不上传隐私
- **跨平台** — Windows / Mac / Linux 都能用
- **轻量快速** — 启动快，占用小

### 🛒 安装

```bash
npm install -g opencut
```
