想换 API 中转商得手改 JSON;五个工具配置格式全不一样,切一次找半天……

**CC Switch** —— 一个开源 App 统一管理 5 个 AI 编程 CLI ✨

---

### 🎯 核心能力

- 一界面管 Claude Code / Codex / Gemini CLI / OpenCode / OpenClaw
- **50+ 供应商预设**(AWS Bedrock、NVIDIA NIM、各家中转),复制 Key 就能用
- 一键切换 + 统一管理 MCP / Prompts / Skills
- 托盘秒切,Claude Code 还支持**热切换**(不重启终端)

---

### 💎 三大亮点

**1️⃣ 用量成本追踪 📊** 跨供应商统计支出、Token,**不怕账单爆雷**。

**2️⃣ 云同步 ☁️** Dropbox / iCloud / 坚果云 / WebDAV,换电脑无感迁移。

**3️⃣ MIT 开源免费 🆓** Tauri + Rust 原生,SQLite 原子写入,配置不损坏。

---

### 🛒 安装

`brew install --cask cc-switch`(macOS) · Win/Linux 去 [Releases](https://github.com/farion1231/cc-switch/releases) 下载
