## 📖 文章正文

与AI共事后的ADHD效应

你以为AI让你效率翻倍？不，它让你变成了多线程机器。

每天的工作：5个AI窗口同时开。ChatGPT写文案，Claude改代码，Copilot补全，Midjourney生图，Gemini读文档。切来切去，大脑过载。

我发现一个残酷真相：越会用AI的人，专注力流失越快。

以前写一篇文章：2小时沉浸。现在：查资料(Claude)→写提纲(GPT)→润色(Claude)→配图(MJ)。每个环节换一个AI，每个切换都在消耗意志力。

你以为你在用AI，其实是AI在用你。

### ADHD效应

- 无法专注超过5分钟
- 习惯性Ctrl+TAB
- 同时开着8个对话
- 每个都没看完
- 越忙越焦虑，越焦虑越碎片

### 3个解药

1. **用长文本模型替代碎片查询** — 一次给足上下文，减少来回切换
2. **任务打包：集中一个AI做完** — 不跨工具，减少切换成本
3. **离线思考：先想清楚再问AI** — 减少无效对话

工具降噪，专注回归。AI应该解放你，不是把你变得更像机器。

## 🧠 核心洞察

"ADHD效应"本质是**工具扩散带来的认知税**：

- 每个AI工具都是一个新的切换点
- 切换成本 ≈ 15分钟重新进入状态
- 越"高效"的工具链 = 越多的上下文碎片
- 最终你变成了一台高速切换的机器，而不是一个深度思考的人

### 为什么越会用AI越累？

| 表面 | 真相 |
|------|------|
| 多工具并行 = 效率高 | 多工具并行 = 注意力分散 |
| AI替我做重复工作 | 管理AI本身就是新工作 |
| 上下文越短越好 | 上下文越碎越耗脑 |

**解药不是不用AI，而是减少AI切换频率。**
