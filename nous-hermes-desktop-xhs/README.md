1. **Jensen GTC 首秀** — NVIDIA CEO 在 GTC 大会上亲自演示 Hermes Desktop
2. **开源桌面 Agent** — MIT 许可，不是网页封装，是跑在本地桌面上的自主 Agent
3. **跨平台** — macOS 12+, Windows 10/11, Linux
4. **核心能力**:
   - 跨会话持久记忆
   - 自然语言定时任务调度
   - 并行子任务代理（Sub-agents）
