# AI搞钱3条路｜普通人也能接的副业

AI副业不是画饼，是实打实能接单的三条路，亲测可行。

## 1️⃣ AI写稿接单

小红书/公众号/知乎代写，一篇50-300元。用Claude写初稿 + 人工润色，效率翻倍。关键是"AI辅助"不是"AI代写"，客户要的是结果不是过程。

**适合人群**：会写作、有网感的人

**接单渠道**：闲鱼、小红书私信、豆瓣小组、朋友圈

**定价参考**：
- 小红书笔记：50-150元/篇
- 公众号推文：100-300元/篇
- 知乎回答：30-80元/条

**工具栈**：Claude 3.5 / ChatGPT → 人工润色 → 配图（Canva/MJ）

## 2️⃣ AI做头像/壁纸/表情包

Midjourney出图，一张头像10-50元，表情包一套88-198元。小红书搜"AI头像定制"，需求量大到离谱。关键是风格统一 + 出图快。

**适合人群**：审美在线、愿意学prompt的人

**接单渠道**：小红书开店、闲鱼、淘宝店合作

**定价参考**：
- 单张头像：10-50元
- 头像+壁纸套装：68-99元
- 表情包一套（16-32张）：88-198元

**工具栈**：Midjourney v6 / DALL-E 3 / Stable Diffusion → Photoshop微调

**关键技巧**：
- 建立风格库（日漫/韩系/美式/国潮）
- prompt模板化，5分钟出4张备选
- 表情包要表情丰富、动作夸张

## 3️⃣ AI做PPT

Gamma / 讯飞智文 / 美图WPS，10分钟出一套PPT。企业汇报/课件/路演，一套100-500元。关键是模板库 + 改稿速度。

**适合人群**：懂业务逻辑、会排版的人

**接单渠道**：闲鱼、小红书、朋友圈、企业外包群

**定价参考**：
- 学生课件：50-150元/套
- 企业汇报：200-500元/套
- 路演BP：300-800元/套

**工具栈**：Gamma / 讯飞智文 → 人工调整 → Canva美化

**关键技巧**：
- 积累行业模板（科技/教育/金融/医疗）
- 数据可视化能力加分
- 改稿控制在2轮内

## 三条路共同点

· 门槛低，但需要"AI + 人"的配合
· 前期靠量积累口碑，后期靠复购
· 别指望躺赚，但月入2k-5k不难
· 选一条深耕，别贪多

## 建议

从最顺手的开始。如果你爱写就选写稿，审美好就做图，逻辑强就做PPT。先接3单练手，再考虑扩大。
