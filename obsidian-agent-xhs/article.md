Obsidian+AI真有搞头

Obsidian 和 AI Agent 到底有没有搞头？
我花了一周研究，结论是：有，而且是大机会👇

其实 Obsidian 之前有个大问题：它是本地的，AI Agent 碰不到。
但最近事情变了。

2026年初 Obsidian 1.12 发布 CLI，搜索比 grep 快54倍。
正式宣告：AI Agent 可以原生读写你的知识库了。

几个关键信号：

1. CLI 打通 + MCP 桥梁
Obsidian CLI + MCP 服务器，AI 直接查 vault
不用再手动复制粘贴到 Prompt

2. Agents Read, Humans Write
AI 读你的笔记做语义搜索、联想、上下文召回
但从不替你写——你的思考留在 Obsidian
这才是正确的分工

3. Claude Code + QMD 实战
QMD 自动导出 Claude 会话到 markdown
关闭即写回 vault，token 省60%+
整个工作流变得丝滑

4. 生态爆发
Copilot · Smart Connections · Claudian · Agent Client
四个插件覆盖聊天/语义/代码/多模型

所以结论是：
AI 不应该在你的笔记里聊天
AI 应该在你写笔记时安静地递材料

Obsidian 不是被 AI 替代
它是被 AI 激活了🔋
