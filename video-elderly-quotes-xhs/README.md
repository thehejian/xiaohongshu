# AI老人语录月入过万｜微信视频号AI语录红利

微信视频号最近出现了一个很火的玩法：用 GPT 模仿莫言、余华等文学大师写语录，配合 AI 生成的老人头像和慢节奏背景音乐，中老年人疯狂转发。一条视频播放量几十万起步。

## 这个套路拆解

### ① GPT 写语录
投喂莫言、余华等文学大师的风格样本，GPT 模仿写作风格，产出哲理人生金句。关键点：
- 投喂 5-10 篇该作家的真实语录作为风格参考
- prompt 引导："用莫言的风格写一段关于人生的感悟"
- 输出要短小精悍，15-30 字一句，适合视频展示

### ② AI 做头像
用 Midjourney 生成"慈眉善目老人"形象：
- prompt 示例：`portrait of a kind elderly Chinese man, warm smile, grey hair, traditional style, photorealistic --ar 3:4 --v 6`
- 真实到中老年用户以为是真人在出镜
- 不同视频用不同头像，避免被识别为 AI

### ③ 慢 BGM
选择抒情轻音乐/钢琴曲，节奏舒缓：
- 中老年审美偏好：舒缓、抒情、怀旧风格
- BGM 音量压低，不要盖过语音
- 推荐：钢琴曲、古筝、二胡等传统乐器

### ④ 视频号分发
微信视频号算法特点：
- 精准触达 45-65 岁人群
- 社交分发：中老年人爱转发给朋友/家人群
- 裂变效应：一条爆了能持续推流 3-7 天

## 为什么能火？

中老年人对"人生哲理"有天然信任感，尤其是"名人名言"：
- **情感共鸣**：金句触发人生感慨，想分享给朋友
- **权威背书**：以为是莫言/余华本人，天然可信
- **社交货币**：转发体现"有文化""有品位"

情感共鸣 + 权威背书 = 高转发率。这是微信生态独有的流量红利。

## 实操建议

1. **起号阶段**：每天发 3-5 条，测试哪类语录数据好
2. **内容差异化**：同一段话换不同头像/BGM 发布
3. **评论区运营**：用"大师说得对"等号评论互动
4. **规避风险**：不要直接写"莫言说"，用"一位老人说"避免侵权
5. **用户心理**：内容长辈看了会"惦记着转发"

## 所需工具

- **Midjourney v6**：生成老人头像
- **剪映**：视频剪辑 + 配 BGM + 字幕
